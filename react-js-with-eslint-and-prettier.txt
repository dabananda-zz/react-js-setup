Follow the steps: https://dev.to/chgldev/getting-prettier-eslint-and-vscode-to-work-together-3678

Replace the settings of .eslintrc with the below settings:
{
  "parserOptions": {
    "ecmaVersion": 12,
    "sourceType": "module",
    "ecmaFeatures": {
      "jsx": true
      }
    },
  "extends": [
    "plugin:prettier/recommended"
  ],

  "plugins": ["prettier"],

  "rules": {
    "prettier/prettier": "error"
  }
}
